你可以通过设置页面的 Frontmatter，在页面禁用功能与布局。

<!-- more -->

本页面就是一个示例，禁用了如下功能:

- 导航栏 ：navbar: false
- 侧边栏 ：sidebar: false
- 路径导航 ：breadcrumb: false
- 页面信息 ：pageInfo: false
- 贡献者 ：contributors: false
- 编辑此页链接 ：editLink: false
- 更新时间 ：lastUpdated: false
- 上一篇/下一篇 链接： prev: false next: false
- 评论 ：comment: false
- 页脚 ：footer: false
- 返回顶部按钮 ：backtotop: false
